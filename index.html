HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • There are 3 main types of content: meta, content, and semantic elements.

CSS

  • 3 main ways to style a webpage: inline, internal, and external style sheet
  • External style sheet is preferred because it's a separate file and easier to maintain
  • Make sure the css file is in the assets folder for it pull in!
  • Using * is like SQL and it will apply to ALL

Git

  • Git is the local branch tool we will use for applications
  • Github is the online branch tool we will use for applications and collaborating with others

JavaScript

  • JavaScript is used to interact with a website
  • Loops are more efficient for arrays
  • Functions perform a specific output when called
